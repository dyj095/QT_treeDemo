# QT_treeDemo
基于QT的QTreeWidget扩展的多级组织结构（多级部门及带圆形头像的人员）

注：运行本Demo需要修改mainwindow.cpp第135行代码中的images目录的绝对路径；

开发环境：
1、QT版本：Qt5.11.0;
2、编译器：MinGW
